# Data Notes

This project uses store-level weekly Walmart sales data with fields such as store number, date, weekly sales, holiday flag, temperature, fuel price, CPI, and unemployment.

## Public Sample

`public_sample_walmart_sales.csv` is a small sample from the clean store-level sales dataset. It does not include personal customer identifiers.

## Large Dataset Note

A separate large retail purchase file in the local folder contains user and product identifiers. That file is not uploaded publicly and is not used as the main evidence for this repository.
