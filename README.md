# Walmart Sales Analytics

Retail analytics project using Python, SQL, and dashboard style reporting to explore Walmart sales performance and business trends.

## Why I Built This

Retail data is a strong way to practise end to end analytics because it connects data cleaning, sales trends, product and store performance, and business recommendations.

## Business Questions

This project looks at sales trends, store performance, product category behaviour, seasonality, and the factors that may influence revenue patterns.

## Tools Used

Python, Pandas, SQL, Jupyter Notebook, Excel, PowerPoint, and business analytics reporting.

## What I Implemented

I prepared and explored sales data, wrote SQL analysis queries, built notebook based analysis, and created reporting material to explain the business story behind the numbers.

## Repository Guide

The notebooks folder is for Python analysis. The sql folder is for query work. The reports folder is for presentation or written outputs. The data folder is for sample data and dataset notes.

## Outputs And Results

Outputs include exploratory analysis, SQL based sales summaries, visual reporting, and a business focused presentation of key findings.

## How Recruiters Should Review This

Review this README first, then check the notebook and SQL files. This project shows practical Python and SQL analytics for a familiar retail business problem.
