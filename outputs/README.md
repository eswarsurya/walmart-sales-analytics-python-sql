# Public Output Evidence

This folder contains public-safe output evidence for the Walmart Sales Analytics project.

## Files

- `dataset_profile.csv` summarises the data size, store count, and date range.
- `top_stores_by_sales.csv` lists the highest-selling stores by total weekly sales.
- `yearly_sales_summary.csv` summarises total sales by year.
- `holiday_sales_comparison.csv` compares holiday and non-holiday sales weeks.
- `sales_driver_correlation.csv` gives simple correlations between weekly sales and economic/weather factors.
- `analysis_output_summary.md` explains the business value of the analysis.
